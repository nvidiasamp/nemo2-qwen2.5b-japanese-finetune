# Details:
创建一个Python脚本 `scripts/01_import_model.py`。该脚本应只执行一次，将模型保存在本地 `./models/qwen25_0.5b` 目录。
```python
# scripts/01_import_model.py
from nemo.collections import llm

# 官方文档验证的正确导入方式
if __name__ == "__main__":  # 官方要求必需
    llm.import_ckpt(
        model=llm.Qwen2Model(llm.Qwen25Config500M()),  # 官方文档配置
        source='hf://Qwen/Qwen2.5-0.5B',
        output_path='./models/qwen25_0.5b.nemo',
        overwrite=False
    )
    print('模型导入完成！')
```

# Test Strategy:
执行脚本后，在 `./models/` 目录下成功生成 `qwen25_0.5b.nemo` 文件，大小约为1-2GB。检查日志无报错。
